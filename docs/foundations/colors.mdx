## Usage

```css
.card {
  background-color: var(--background);
  color: var(--foreground);
  border: 1px solid var(--border);
}
```

```tsx
<div className="bg-primary text-white">Blood Orange CTA</div>
<div className="bg-background text-foreground">Themed surface</div>
```
