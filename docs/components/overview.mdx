# Components

Browse all available UI components in the Stacks Design System.

Each component is designed with accessibility, customization, and Web3 use cases in mind. Click on any component to view detailed documentation, examples, and API reference.

| Component | Description |
|-----------|-------------|
| [Button](./button) | Triggers an action or event, such as submitting a form |
| [Card](./card) | Container for grouping related content and actions |
| [Dialog](./dialog) | A modal window that overlays the main content |
| [Input](./input) | A form input for collecting user data |
| Accordion | A vertically stacked set of interactive headings that reveal content |
| Alert | Displays important messages to users with contextual styling |
| Avatar | An image element with a fallback for representing users |
| Badge | Small status descriptor for UI elements |
| Context Menu | Displays a menu triggered by right-click or long-press |
| Dropdown Menu | Displays a list of actions triggered by a button |
| Slider | An input for selecting a value from a range |
| Tabs | A set of layered sections of content displayed one at a time |
